# Zerodha Clone

This repository contains a frontend and backend for the "Zerodha Clone" project.

Quick steps to create a remote and push:

1. Create a GitHub repo (via website or `gh` CLI).
2. Add remote and push:

```
git remote add origin https://github.com/USERNAME/REPO.git
git branch -M main
git push -u origin main
```

Or with the GitHub CLI:

```
gh repo create USER/REPO --public --source=. --remote=origin --push
```

Files added locally: `.gitignore` and initial commit.
